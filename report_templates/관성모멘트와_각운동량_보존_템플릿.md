```
각각의 가속도의 평균값을 관성모멘트 공식
$$I = \\frac{mg \\cdot r^2 - m \\cdot r \\cdot a \\cdot t^2}{a}$$

```

---

#### 관성모멘트 계산

**강체 조합별 관성모멘트:**
- I (Shaft & Pulley만)
- (I + I_A) (Shaft + Disk A)
- (I + I_A + I_B) (Shaft + Disk A + Disk B)
- (I + I_A + I_R) (Shaft + Disk A + Ring)

**[LLM 작성]** 계산:
```
이때 회전축과 도르래의 관성모멘트를 I, 
Disk A의 관성모멘트를 I_A, 
Disk B의 관성모멘트를 I_B, 
Ring의 관성모멘트를 I_R이라 하면 
각각의 관성모멘트를 다음과 같이 구할 수 있다.

I = {{I_shaft}}
I_A = (I_A+I) - I = {{I_A+I}} - {{I}} = {{I_A}}
I_B = (I_B+I_A+I) - (I_A+I) = {{I_B+I_A+I}} - {{I_A+I}} = {{I_B}}
I_R = (I_R+I_A+I) - (I_A+I) = {{I_R+I_A+I}} - {{I_A+I}} = {{I_R}}
```

---
