## 5. 실험 결과 분석

### 이론 배경

**일-에너지 정리:**
$$W_T = \\Delta K_{cart}$$

줄이 카트에 해준 일 $W_T$와 카트의 운동에너지 변화량 $\\Delta K_{cart}$가 같아야 한다.

**역학적 에너지 보존:**
$$W_{mass} = \\Delta K_{system}$$

추의 무게가 한 일 $W_{mass}$와 계의 운동에너지 증가량 $\\Delta K_{system}$이 같아야 한다.

1. **비보존력의 존재**
   - 비보존력이 있으면 계의 역학적에너지가 보존되지 않음
   - 마찰력이 작은 바닥 사용, 진공상태에서 실험

2. **힘 센서의 측정 오류**
   - 로드셀의 변형에 의한 저항변화가 지속적으로 발생
   - 더 정밀한 측정기 사용

   - 무선 힘 센서 사용

4. **초기 속력**
   - 측정이 바로 시작되지 않아 초기 속력이 0이 아님
   - 더 정밀한 시간 측정장치 필요

5. **ΔK_cart > W_T**
   - 마찰력이 운동에너지를 감소시키기 때문
   - 마찰력을 고려하면 일치할 것
