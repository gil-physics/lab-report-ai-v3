$$R = \\frac{v_0^2 \\sin(2\\theta)}{g}$$

**오차율 계산:**
$$오차율(\%) = \\frac{R_{avg} - R_{이론값}}{R_{이론값}} \\times 100$$

### 궁금증

**[LLM 작성]**:

**Q1: Free Fall 실험에서 평균 a를 구할 때 평균 속도를 먼저 구하면 안될까?**
```
안될 것이다. 실험에서 구한 방식은 각각의 trial에서 a를 구했기 때문에 
속도의 변화량이라는 가속도의 정의와 맞지만, 속도끼리 평균을 구하고 가속도를 구하면 
정확한 속도의 변화량이라고 할 수 없기 때문에 실험 방식이 더 정확하다.
```

**Q2: Projectile Motion 실험에서 왜 가속도로 오차를 분석하지 않고 거리로 분석할까?**
```
이론에서 이동거리와 중력가속도는 단순 반비례관계이므로, 
오차율을 구하는 방식이 같다면 오차율은 같게 나온다.
```

**Q3: 30°와 60°, 15°와 75°의 사거리가 이론적으로 같아야 하는데 왜 다른가?**
```
오차 확인에서 알아본 운동시간에 따른 공기저항 등의 차이가 영향을 줬을 것이다.
```
